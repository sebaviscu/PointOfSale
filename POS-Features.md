
# Point of Sale

## Features

### Admin

* Clientes:
  * ABM de clientes
* Gastos:
  * ABM de gastos
  * ABM tipos de gastos
* Proveedores:
  * ABM proveedores
  * ABM pago a proveedores
* Turnos:
  * Solo muestra turnos
  * Se pueden cargar por excel
* Users:
  * ABM usuarios

### Inventario

* Productos:
  * ABM productos
  * Imprimir precio
* Promociones:
  * ABM promocion

### Configuracion

* Categorias:
  * ABM de categorias
* Formas de pago:
  * ABM formas de pago
* Tiendas:
  * ABM de tiendas (sucrusales)

### Reportres

* Reporte productos
* Reporte ventas

### Web

* Shop:
  * Tienda online
* Ventas web:
  * Baja y modificacion de ventas web

### Venta

* Alta de venta
